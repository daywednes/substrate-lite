[−][src]Module substrate_lite::chain::sync::full_optimistic

Optimistic header and body syncing.

This state machine builds, from a set of sources, a fully verified chain of blocks headers and bodies.

In addition to managing the sources, using OptimisticFullSync also requires holding the storage of the latest finalized block.
